    Керування локальними фотографіями: Шукати/Копіювати/Змінювати-Exif та показувати в Галереї або на Карті.

    Особливості:

    * Швидко знаходьте фотографії за тегами(ключові слова), (під-)папку, дату, географію, назву, ...
    * Відображення результатів у галереї або географічній карті з openstreetmap.
    * Функція детального перегляду збільшення масштабу, переміщення до наступного/попереднього зображення.
    * Вбудований файловий менеджер для фотографій: пошук, сортування, перегляд, копіювання, видалення, надсилання ... .
    * Редагування метаданних exif: дати, заголовку, опису, тегів(ключових слів), географії, рейтингу, ....
    * Автоматична обробка на основі правил-папки: фотографії автоматично перейменовуються та отримують теги, географічні назви, назви ... коли ви копіюєте/переміщуєте їх
    *Позначте зображення як "ПРИВАТНЕ", щоб приховати їх від інших галереї-додатків та збирачів зображень.
    * У "Захищеному/закріпленому режимі" потрібні небезпечні команди для додатків, такі як редагування/копіювання/видалення/спільне використання/налаштування/"зміни вибору зображення" вимкнено, щоб ви могли передати телефон комусь іншому
    * Можна обробляти великі колекції зображень (20000+ зображень в 1000+ папках).
    * Використовує контент-провайдер зображення для Android. Немає необхідності в скануванні початкового зображення.
    * Розширений медіа-сканер для фото Exif, IPTC, XMP

    Необхідні дозволи для Android:

    * INTERNET: для завантаження данних карт з OpenStreetmap Server
    * ACCESS_NETWORK_STATE and ACCESS_WIFI_STATE: щоб дізнатись, чи доступний Wi-Fi / Інтернет, щоб розпочати завантаження геоданих
    * WRITE_EXTERNAL_STORAGE для кешування завантажених даних карти в локальній файловій системі та для виконання файлів з фотографіями
    * READ_LOGS щоб читати та зберігати потенційні аварійні журнали в текстовому файлі.
    .
